# 图片资源目录

请将以下图片文件放置在此目录中：

## 需要的图片文件

1. **doctor-office.jpg** - 医生办公桌/接诊照片（用于首页 Hero 区域）
2. **waiting-area.jpg** - 等候区与中式座椅照片
3. **treatment-room-1.jpg** - 诊疗床与针灸设备照片
4. **treatment-room-2.jpg** - 私人诊疗室照片
5. **herb-cabinet.jpg** - 中药柜与办公区照片

## 图片建议规格

- **格式**：JPG、PNG 或 WebP
- **尺寸**：建议宽度 1200px 以上，以获得更好的显示效果
- **文件大小**：建议每张图片压缩到 500KB 以下，以提升页面加载速度

## 使用说明

所有图片路径已在 `index.html` 中配置为 `images/文件名.jpg` 格式。
直接将图片文件放入此目录即可。
